Steps for pushing changes made in local folder to remote repository
Using terminal or git bash, cd into dsi35-projects folder and run the following commands.
git add . # add files to be tracked
git commit -m “your commit message” # commit staged files
git push # push changes in the git repository to the remote repository (github.com)
Steps for pulling changes made on remote repository to local folder
git pull
